Automatizace — GitHub Actions

  • 1
    Každý den v 9:00 SEČ se automaticky spustí GitHub Actions workflow (update.yml)
  • 2
    Skript stáhne aktuální ceník z Mamtex.cz (přes zabezpečenou URL s hashem)
  • 3
    Vypočítá marže pro každý produkt a uloží výsledek do marze_export.csv
  • 4
    Commitne a pushne změněný soubor na GitHub s názvem commitu Aktualizace YYYY-MM-DD
  • 5
    Feed je dostupný na raw GitHub URL — Napojse.cz si ho přečte

Výpočet marže

Vstupní sloupce z Mamtexu

  • purchasePrice — nákupní cena
  • standardPrice — prodejní cena
  • code — kód produktu
  • name — název produktu

Vzorce

Marže Kč = standardPrice − purchasePrice

Marže % = (Marže Kč / standardPrice) × 100

Multishop = Marže % zaokrouhlená na 5

Zaokrouhlení Multishop

  • 27% → 25% (dolů — zbytek 2)
  • 28% → 30% (nahoru — zbytek 3)
  • 51% → 50% (dolů — zbytek 1)
  • 53% → 55% (nahoru — zbytek 3)

Pravidlo: standardní zaokrouhlení na nejbližší násobek 5 (zbytek 0–2 → dolů, zbytek 3–4 → nahoru).
